# OneFlow Frontend

React + Vite + Tailwind CSS frontend for OneFlow project management system.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Technologies

- React 18
- Vite
- React Router
- Tailwind CSS
- Axios
- Lucide React (icons)
- Recharts (charts)

## Project Structure

- `/src/components/common` - Reusable UI components
- `/src/components/layout` - Layout components (Navbar, Sidebar)
- `/src/pages` - Page components
- `/src/context` - React Context providers
- `/src/utils` - Utility functions and API client
